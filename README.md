## 🛡 Safety and Control

AgentManager prioritizes safe operation. The kill switch lets you stop everything instantly if needed. The app monitors agent actions to prevent runaway processes. This reduces risks while running many agents at once.

## 🆘 Troubleshooting

Here are tips if you run into problems:

- **AgentManager won’t open:** Restart your computer or try reinstalling from the releases page.  
- **Agents won’t start:** Check your internet connection and try again.  
- **High costs:** Use the cost tracking panel to identify which agents consume most resources. Stop them if needed.  
- **App freezes:** Use the kill switch to stop all agents, then restart the program.
